# Beta Known Issues Register

## Required Fields

| Field | Value |
| --- | --- |
| Issue ID | pending |
| Severity | pending |
| Summary | pending |
| Owner | pending |
| Workaround | pending |
| Blocks Beta | pending |
| Accepted For Beta | pending |
| Evidence Path | pending |

## Required Rules

- known issue ID is required
- known issue summary is required
- known issue owner is required
- high and critical known issues must block beta or be explicitly accepted
- accepted beta known issue requires workaround
- known issue evidence path must live under docs/beta_launch/

## Boundary

This register records known issue governance. It does not accept unresolved issues automatically.
