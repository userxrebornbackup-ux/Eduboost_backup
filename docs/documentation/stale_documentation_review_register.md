# Stale Documentation Review Register

## Required Fields

| Field | Value |
| --- | --- |
| Finding ID | pending |
| Path | pending |
| Days Stale | pending |
| Owner | pending |
| Severity | pending |
| Action Required | pending |
| Blocks Release | pending |

## Required Rules

- days stale cannot be negative
- stale documentation owner is required
- release blocker stale docs must block release
- stale documentation path must live under docs/
- documentation review evidence must be retained

## Boundary

This register records stale-documentation review readiness. It does not update documentation automatically.
