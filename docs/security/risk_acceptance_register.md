# Risk Acceptance Register

## Required Fields

| Field | Value |
| --- | --- |
| Risk ID | pending |
| Severity | pending |
| Reason | pending |
| Owner | pending |
| Approver | pending |
| Expiry Days | pending |
| Compensating Controls | pending |
| Evidence Path | pending |

## Rules

- critical risks cannot be accepted for production by default
- accepted risks must expire
- accepted risks require compensating controls
- accepted risks require approver and owner
- risk acceptance evidence must be retained

## Boundary

This register records risk-acceptance governance. It does not approve risk automatically.
