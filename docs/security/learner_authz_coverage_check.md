# Learner Authorization Coverage Check

## Purpose

`script/check_learner_authz_coverage.py` blocks newly introduced learner-scoped
routes that do not contain an authorization or authentication marker.

## Make Target

```bash
make learner-authz-check
```

The check uses the learner authorization matrix generated by:

```bash
make learner-authz-matrix
```

## Verification

```bash
pytest -c pytest.ini tests/unit/test_check_learner_authz_coverage.py -q --no-cov
make learner-authz-check
```
