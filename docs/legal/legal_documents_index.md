# Legal document index

This repository does not contain final legal advice. The following documents must be drafted, reviewed, versioned, and accepted before public beta.

| Document | Owner | Required before |
|---|---|---|
| Terms of Service | Legal/Product | Public beta |
| Privacy Policy | Legal/Compliance | Public beta |
| Child-friendly Privacy Notice | Legal/Compliance/Education | Public beta |
| Parent Consent Notice | Legal/Compliance | Processing real learner data |
| Acceptable Use Policy | Product/Legal | Public beta |
| Refund Policy | Product/Finance/Legal | Paid plans |
| School Data Processing Agreement | Legal/Compliance | Institutional deployments |

## Acceptance tracking

Every policy acceptance must store policy key, semantic version, effective date, actor ID, acceptance timestamp, and source channel. Material updates require notification and renewed acceptance where legally required.
