# Production Readiness Baseline Boundary Contract

## Required Baseline Boundaries

- included
- deferred
- excluded
- external manual

## Required Boundary Fields

- boundary ID
- area
- title
- boundary
- rationale
- evidence path
- owner
- manual dependency where applicable

## Required Rules

- baseline boundary title is required
- baseline boundary rationale is required
- baseline boundary evidence path must be controlled
- external/manual boundary requires manual dependency
- repository-side evidence must not be represented as live external approval

## Boundary

This contract records the boundary between repository evidence and future/external work.
