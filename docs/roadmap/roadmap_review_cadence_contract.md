# Roadmap Review Cadence Contract

## Required Cadence Fields

- cadence ID
- frequency days
- owner
- required inputs
- required outputs
- evidence path
- blocks scope expansion

## Required Inputs

- beta feedback
- known issues
- security findings
- support metrics
- operations metrics

## Required Outputs

- updated roadmap register
- deferred scope register
- risk register
- owner assignments

## Boundary

This contract records roadmap review cadence. It does not run review meetings.
