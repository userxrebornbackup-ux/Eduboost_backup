# Production Learner and Parent UX Contract

## Purpose

This contract records production-readiness expectations for learner and parent/guardian UX.

## Required Learner UX

- Complete learner dashboard.
- Show study plan.
- Show next recommended lesson.
- Show progress.
- Show streak if gamification enabled.
- Show weak topics.
- Show recommended next activity.
- Complete diagnostic question display.
- Complete diagnostic progress indicator.
- Complete diagnostic answer submission.
- Complete diagnostic result summary.
- Complete lesson explanation view.
- Complete worked example view.
- Complete practice question interaction.
- Complete hints.
- Complete answer reveal.
- Complete feedback capture.
- Complete report-content issue flow.
- Add pause/resume diagnostic UX.
- Add offline-friendly lesson state.
- Add learner personalization settings.

## Required Parent/Guardian UX

- Complete parent dashboard.
- Show child progress.
- Show consent status.
- Show recommended support actions.
- Show reports.
- Show privacy controls.
- Add data export request UI.
- Add erasure request UI.
- Add data correction request UI.
- Add processing restriction request UI.
- Add subscription/billing UI.
- Add consent renewal UI.
- Add notification preferences UI.
- Add weekly parent report view.
- Add how to help at home guidance.
