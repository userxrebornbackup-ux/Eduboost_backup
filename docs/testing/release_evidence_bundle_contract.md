# Release Evidence Bundle Contract

## Purpose

This contract defines release evidence bundle requirements.

## Required Bundle Items

- test report
- coverage report
- security scan
- accessibility report where applicable
- performance report where applicable
- OpenAPI artifact
- release approval
- smoke test report
- known issues register
- rollback review
- deployment gate output
- release owner signoff

## Required Bundle Fields

- evidence ID
- evidence type
- path
- generated by
- Git SHA
- checksum SHA-256
- required release stage
- retained flag

## Boundary

This contract records release evidence bundle readiness. It does not approve release promotion.
