# Known Issues Release Register

## Required Fields

| Field | Value |
| --- | --- |
| Issue ID | pending |
| Severity | pending |
| Owner | pending |
| Release Stage | pending |
| Fix or Waiver | pending |
| Waiver Expiry | pending |
| Customer Impact | pending |
| Evidence Link | pending |

## Rules

- release blockers cannot be open for production
- critical issues require owner and fix or approved waiver
- known issues review is required before beta and production
- known issues register must be retained in release evidence bundle

## Boundary

This register records known issue governance. It does not approve release with open issues.
