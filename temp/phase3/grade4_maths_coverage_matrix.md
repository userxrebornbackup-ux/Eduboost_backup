## Human Review Checklist

Use the following checklist for every item during P3-05 through P3-07.

- [ ] Can a Grade 4 learner in South Africa understand this question without teacher assistance?
- [ ] Is the correct answer unambiguously correct — could any expert disagree?
- [ ] Does each distractor represent a real mistake a Grade 4 learner commonly makes?
- [ ] Is the explanation written in language a 9–10 year old can follow?
- [ ] Does the item reference a real CAPS learning outcome?
- [ ] Is there any harmful content, real-world PII, or culturally inappropriate material?

Items failing any check → return to `draft` status.
